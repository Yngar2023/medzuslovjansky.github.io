# Przyimki

## Spójniki \{#conjunctions}

Spójnik jest częścią mowy, który łączy dwa słowa, zdania, wyrażenia lub zdania składowe. Spójniki są nieodmienne i stoją pomiędzy elementami, które łączą. Są trzy rodzaje spójników: _spójniki współrzędne_ łączą dwa lub więcej elementów tej samej ważności, _spójniki skorelowane_ to spójniki występujące w parach oraz _spójniki podrzędne_ łączą zdanie podrzędne ze zdaniem nadrzędnym. Poniżej (niepełna) lista:

### spójniki skorelowane \{#correlative-conjunctions}

**`čim` ... `tym` ...** „czym ... tym ...”

**`ili` ... `ili` ...** „albo ... albo ...”

**`kako` ... `tako i` ...** „jak ... tak i ...”


**`koliko` ... `toliko` ...** „ile ... tyle ...”

**`ne jedino` ... `ale takože`** „nie tylko ... ale jeszcze ...”

**`ni` ... `ni` ...** „nie ... ani ...”
